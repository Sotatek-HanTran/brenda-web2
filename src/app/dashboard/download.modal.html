Download S3 Files

Unfortunately, there is no convenient way to download all S3 files within a bucket from the browser. Here are some potential alternatives to allow you to do so however:

AWS CLI

The AWS Command Line Interface (CLI) is a Python utility built by Amazon. Installation instructions are available to walk you through the installation process. Once the tool is installed and configured, the following command will download all the files in your {{bucket}} bucket to your current folder

aws s3 sync s3://{{bucket}} .

s3cmd

s3cmd is another command line tool that is Python based. Once you have it installed and configured, you may use the following command to download all files to your current folder

s3cmd sync s3://{{bucket}}/ .
