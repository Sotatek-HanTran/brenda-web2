Configure S3 CORS

In order to view the contents of an S3 bucket, CORS needs to be enabled first. Follow the following steps in order to turn CORS on for your buckets. For more information on CORS and S3, the Amazon S3 CORS Documentation has additional information.

  1. Go to the S3 home page (Log in to AWS if necessary)
  2. Highlight and click to select the desired bucket
  3. If not already open, click the "Properties" button to open the bucket properties panel
  4. Expand the "Permissions" section and click on "Add CORS Configuration"
  5. In the dialog that opens, paste the following configuration: 
    <CORSConfiguration>
    <CORSRule>
        <AllowedOrigin>*</AllowedOrigin>
        <AllowedMethod>GET</AllowedMethod>
        <MaxAgeSeconds>3000</MaxAgeSeconds>
        <AllowedHeader>*</AllowedHeader>
    </CORSRule>
</CORSConfiguration>
  6. Click "Save"
Buckets refresh periodically on their own so should update shortly if settings were correct
