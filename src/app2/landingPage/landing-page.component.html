Brenda-Web

Brenda-web is intended to be a web based UI to replace the command line tools provided by the Brenda project.

Setup Your Project! or Follow the Tutorial

Fully Client Side

The core principle of this tool is to be fully client side meaning credentials are not stored anywhere other than in your own browser reducing the risk of them falling into the wrong hands.

Your Own Account

Using your own AWS account means your project files never go to a 3rd party which helps protect your work and gives you ultimate control

Everything in One Place

Everything you need to run and monitor your rendering farm is in one convenient and easy to use tool

Mobile Friendly

Setup and monitor your render jobs from a computer or on the go from your phone or tablet

An Open Source Project

Free to use - you only pay for the AWS resources you use and all source code is available on GitHub

How it Works

Brenda-Web has four primary components:

S3 Source
Location of the Blender project
Work Queue
List of tasks to be processed
Brenda Worker
EC2 instance running Brenda
S3 Destination
Location of finished frames

Monitor Progress

The dashboard allows you to monitor the progress of your jobs

Work Queue

See number of tasks remaining and get estimates of time and cost to completion

Workers

Visibility into each worker's state, how it is performing and the ability to quickly terminate workers

Destination Bucket

Get a list of all the frames that have been completed and quick access to preview or download frames
